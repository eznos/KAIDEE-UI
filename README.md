# BMS-Back-Office-UI

## User interface for Kaidee System

### Project setup

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run serve
```

### Compiles and minifies for production

```bash
npm run build
```

### Lints and fixes files

```bash
npm run lint
```

### Deploy app to firebase hosting

```bash
firebase deploy --only hosting
```
